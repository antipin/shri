Опишите своими словами, что происходит в браузерах при клике на ссылку yandex.ru (полнота ответа — на ваше усмотрение).
  1. Если в браузере включен javascript:
    1. Если javascript-событие click кто-то слушает, выполняем соответствующие функции в порядке подписки на событие.
    2. Если какая-то из функций, подписанных на событие click, вызывает event.preventDefault() или возвращает false, действие по умолчанию не выполняется.
  2. Выделяем URL из атрибута href
  3. Парсим URL на составные части, определяем валидность.
  4. Если URL-схема может быть обработана браузером, продолжаем, иначе ищем приложение, отвечающее за обработку данной схемы.
  5. Если приложение найдено, запрашиваем подтверждение открытия приложения у пользователя и передаем управление приложению.
  6. Если URL-схема может быть обработана браузером (в случае с HTTP):
    1. Если URL равен текущей страницеь, но задан fragment, переходим к элементу с id = fragment
    2. Отправляется GET запрос
    3. Ожидаем ответ
    4. Парсим, выделяем ссылки на CSS, javascript и т.д., запрашиваем их.
    5. Строим DOM
    6. Рендерим страницу
